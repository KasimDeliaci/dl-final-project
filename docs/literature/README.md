# Literature Notes

Bu klasör literatür taraması için ayrılmıştır.

Şu aşamada içerik doldurulmamıştır. Amaç, GPT Pro/deep research veya manuel okuma sonrası kaynakları kontrollü biçimde yerleştirecek scaffold'u hazır tutmaktır.

Önerilen yapı:

```text
docs/literature/
├── literature_index.md
├── paper_registry.yaml
├── papers/
├── notes/
└── axes/
```

Kurallar:

- PDF dosyaları local convenience copy olabilir; büyük dosyaları Git'e almadan önce dikkatli olun.
- Her paper için source URL, citation metadata ve projedeki rol `paper_registry.yaml` içinde izlenmelidir.
- Literatür notları training code tarafından parse edilmemelidir.
