# Scripts Directory

Bu klasör ileride command-line runner script'leri için ayrılmıştır.

Şimdilik kod eklenmemiştir. Script'ler notebook içine gömülü logic yerine reproducible çalıştırma noktaları olmalıdır.

Beklenen ilerideki script rolleri:

- dataset audit,
- feature extraction,
- MLP training/evaluation,
- fusion experiment runner,
- fine-tuning runner,
- report figure generation.
